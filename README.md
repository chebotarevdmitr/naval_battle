# 🚢 Морской бой — Qt Quick / Android (в разработке)

> «Теперь корабли не просто буквы — они… оживают!»  
> — Дмитрий, после первой анимации взрыва.

Этот проект — **развитие классической игры «Морской бой»** с использованием:

- **Qt 6** — современный C++ фреймворк,
- **Qt Quick (QML)** — для плавных анимаций и эффектов,
- **Android** — сборка под мобильные устройства.

## 🧪 Текущий статус

- Ветка: `feature/qt-quick-android`
- Стабильная версия: `v1.0` (графическая, на QWidget)
- Эта ветка — **экспериментальная**, под анимации и мобильные устройства.
